FMC Host Clock Output Device¶

Authors

Jonathan P. Newman

IO

Register Access

ONIX ID

22

ONIX Hubs

PCIe Host

Description¶

The FMC Host Clock Output device generates a clock output that is synchronized to a local hardware clock. It has very fast rising edges and a high-current output that can drive 50 ohm coaxial cable.

The output of this clock can optionally be gated by the ONI-specified RUNNING gate to link the clock to the start of data acquisition within a given Acquisition Context.
