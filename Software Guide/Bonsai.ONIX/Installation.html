Installation¶

Bonsai.ONIX is a Bonsai library for ONIX hardware and can be installed using its package manager.

  1. First you will need to install Bonsai

  2. Open Bonsai. When you are greeted by the start menu, click “Manage Packages.”

  3. Set the package source to “Bonsai Community”.

  4. Search for “ONIX” and install. Installing Bonsai.ONIX.Design will install both the base Bonsai.ONIX library and its visualization tools.

    Tip

    It is recommended to also install Bonsai.ONIX.Design because it provides a substantial set of data visualation and graphical hardare configuration elements.
